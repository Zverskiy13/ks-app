# Клиники Столицы — приложение (обзор для ревью)

Личное корпоративное веб-приложение (PWA) владельца группы клиник «Клиники Столицы».
Один Python-сервис (FastAPI) отдаёт и REST API `/api/*`, и сам фронтенд (PWA) из `webapp/`.
Ставится на iPhone/Android «на экран Домой», работает офлайн (service worker), шлёт web-push.

## Архитектура и стек
- **Backend:** Python 3.13 / FastAPI (`app.py`, ~2200 строк). Запуск: `uvicorn app:app` (см. `Procfile`).
- **Frontend:** ванильный JS PWA в `webapp/` — `index.html`, `app.js` (весь UI, объект `RENDER` по экранам), `api.js` (обёртки REST), `styles.css`, `sw.js` (service worker, network-first для кода), `manifest.webmanifest`, `icons/`.
- **Хранилище состояния:** файлы JSON/MD в отдельном приватном GitHub-репозитории через GitHub API
  (переменные `GITHUB_TOKEN` / `GITHUB_REPO` / `GITHUB_BRANCH`). Кэш чтений в памяти (`_GH_CACHE`, TTL ~25с).
  Есть безопасное чтение `load_store_safe` (защита от затирания при сбое чтения) и `gh_delete`.
- **Хостинг:** Railway (сервис `web`, авто-деплой из GitHub).
- **Аутентификация:** вход по PIN (`APP_PINS`), роли owner / head / staff, фильтрация по компаниям.

## Разделы приложения
- **Главная** — пульс дня, ИИ-директор (сводка), карточки разделов, рабочий стол (привычки/вес/чекап).
- **День** — календарь-месяц со статусами (зелёный «всё сделано» / красный «есть невыполненное»),
  почасовая сетка задач; блоки поддерживают **повторы** (каждые N дней/недель/месяцев или по датам),
  отметку «выполнено» с возвратом в работу, перенос/правку.
- **Задачи** — приоритеты, сроки, вкладка «Срочные», правка/дедуп.
- **Финансы** — прибыль по направлениям группы; **Агрегатор — маржа** (ежедневная выручка/себестоимость/
  маржа по регионам и клиникам; данные тянет агент из .xls-отчёта на почте по IMAP + парсер `xlrd`).
- **Здоровье** — напоминания о чек-апах, результаты анализов, **распознавание бланков фото/PDF через Claude
  Vision** (режим `labs` → подтверждение → сохранение), файлы на сервере (гибрид), **графики по показателям**
  (свой SVG), **ИИ-рекомендации** (образ жизни / вопросы врачу / пересдача — без диагнозов).
- **Трекер дня** — привычки, вредные привычки со стоимостью, вес с целью, вечерний отчёт, геймификация.
- **Цели, План недели, Дневник, ИИ-помощник, Пилот ПВЛ** (владельцу — данные из Telegram-бота).
- **Уведомления (центр)** — настраиваемые web-push по направлениям (см. ниже).

## Ключевые группы эндпоинтов (app.py)
- Задачи/сетка: `/api/tasks*`, `/api/agenda/add`, `/api/reminders/add`, `/api/item/{done,undone,move,delete,edit}`, `/api/day`, `/api/month`.
- Дедлайны/цели/привычки/неделя: `/api/deadlines*`, `/api/biggoals*`, `/api/habits*`, `/api/weekplan`, `/api/goals`.
- Финансы/агрегатор: `/api/finance/{ingest,agg,pull}`, `/api/group*`.
- Здоровье: `/api/vision` (режимы text/finance/labs), `/api/health/{file,file/delete,advice,checkups}`.
- ИИ/голос/документы: `/api/assistant`, `/api/brain`, `/api/stt`, `/api/vision`.
- Пуши/уведомления: `/api/push/{key,subscribe,test}`, `/api/notif/settings`.
- Пилот ПВЛ: `/api/pvl/{team,report}`.

## Центр уведомлений
Настройки `state/notif_settings.json` (мастер-выключатель + каналы {on, time, days}):
утренний бриф, вечерний отчёт, дедлайны (за N дней), просроченные задачи, план дня,
чек-апы/анализы (за N дней), «агрегатор не обновился». Движок `_daily_notifications()` в фоновом
цикле (проверка раз в минуту, время по МСК, дедуп маркером на день). Данные здоровья лежат на устройстве,
поэтому на сервер синкается только график чек-апов {название, дата} — медицинские значения не покидают телефон.

## Переменные окружения
`GITHUB_TOKEN`, `GITHUB_REPO`, `GITHUB_BRANCH` — хранилище состояния.
`ANTHROPIC_API_KEY` — ИИ-помощник, умный разбор, распознавание анализов, советы.
`OPENAI_API_KEY` — распознавание голоса (Whisper).
`VAPID_PRIVATE` (+ публичный в коде) — web-push.
`APP_PINS` — PIN-коды и роли. `MAIL_USER` / `MAIL_APP_PASSWORD` — агент агрегатора (IMAP).
Секреты — только в окружении, в коде их нет.

## Как запустить локально
```bash
pip install -r requirements.txt
uvicorn app:app --reload   # http://127.0.0.1:8000
```
Без переменных окружения часть функций (ИИ, пуши, хранилище) работать не будет — это нормально для ревью кода.
